# issue-analysis-flow

Статус: project-local flow

## Назначение

Этот документ задает project-specific настройки flow анализа issue для данного
репозитория.

Он предназначен для адаптации общего системного `issue-analysis-flow` под
конкретный проект.

## Как использовать

- опиши project-specific ожидания к анализу issue
- дополни требования к плану реализации
- зафиксируй обязательный контекст, который агент должен учитывать
- не удаляй базовые инварианты системного flow без явного осознанного решения

## Project-Specific Context

Заполни этот раздел:

- какие типы задач здесь встречаются
- какие артефакты анализа обязательны
- какие ограничения проекта нужно учитывать
- какие архитектурные документы агент обязан читать

## Project-Specific Questions Policy

Заполни этот раздел:

- какие вопросы считаются блокирующими
- какие допущения агент может делать сам
- какие темы всегда требуют уточнения у владельца репозитория

## Project-Specific Plan Requirements

Заполни этот раздел:

- какие разделы обязательно должны быть в плане
- какие риски должны быть отражены явно
- какие критерии готовности плана действуют в этом проекте
